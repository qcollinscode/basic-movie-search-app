# Basic Movie Search App

### Live Demo: https://basic-movie-search-app.herokuapp.com/

## Installation

Git clone this repository:

```
git clone https://github.com/qcollinscode/basic-movie-search-app.git
```

Change directory into the cloned repository:

```
cd basic-movie-search-app
```

Install the dependencies that are listed in the package.json file:

```
npm install
```

## Running

Start the application:

```
npm start
```
